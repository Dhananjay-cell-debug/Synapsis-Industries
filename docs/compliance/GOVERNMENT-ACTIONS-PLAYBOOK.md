## 1. LUT Filing (annual)

**When:** Day GSTIN arrives, then every 1 April for the new FY.
**Where:** [GST Portal](https://www.gst.gov.in)
**Time needed:** 10 minutes.
**Cost:** Free.
**Why:** Allows zero-rated export invoices to international clients without collecting IGST. See `TAX-STRATEGY.md#lever-l2`.

### Prerequisites
- GSTIN active.
- Login credentials for GST portal.
- 2 witnesses' details (name, address, occupation). For solopreneur: any 2 trusted adults (friends/family). Their PAN not required, just identity.

### Steps
1. Login at [www.gst.gov.in](https://www.gst.gov.in) using GSTIN + password.
2. Navigate: **Services → User Services → Furnish Letter of Undertaking (LUT)**.
3. Select Financial Year (the current FY).
4. Tick the three self-declaration boxes:
   - I shall not export the goods or services without payment of integrated tax.
   - I undertake to pay integrated tax along with interest...
   - The benefit of bond cum LUT will not be misused.
5. Enter witness details (Name, Occupation, Address × 2 witnesses).
6. Enter place of filing → "Mumbai" (or your principal place of business).
7. Save → Preview → Sign with DSC or EVC (OTP on registered mobile/email).
8. Submit. Download the acknowledgement PDF.

### After filing
- Save PDF to Document Vault → `{fy}/lut/lut_{fy}.pdf`.
- Update `business_tax_profile.lut_active = true`, `lut_filed_at = today`, `lut_reference = ARN_from_PDF`.
- System auto-creates next renewal reminder for 15 March of next FY.

### Common errors
- **"Witness PAN required"** → Some portal versions ask. Use any valid PAN of a trusted adult.
- **DSC not registered** → For sole proprietor, DSC is optional; EVC (OTP) works fine.

## 2. GSTR-1 Filing (quarterly under QRMP)

**When:** 13th of month following quarter end (so for Apr-Jun quarter → 13 Jul).
**Where:** [GST Portal](https://www.gst.gov.in) → Returns Dashboard.
**Time needed:** 15-30 minutes if data is ready.
**Why:** Reports outward supplies (invoices issued) to government. Lets your clients claim ITC against your invoices.

### Prerequisites — generated by the system
The system's **GST Workspace** prepares a downloadable JSON or Excel matching GSTR-1 schema. Sections covered:
- **B2B** — invoices to Indian clients with GSTIN.
- **B2C (Large)** — to Indian unregistered clients > ₹2.5L invoice value.
- **B2C (Small)** — consolidated state-wise summary < ₹2.5L.
- **Export** — international invoices (with shipping bill / FIRC reference).
- **HSN summary** — SAC-wise summary of services rendered.

### Steps
1. Login → **Services → Returns → Returns Dashboard**.
2. Select financial year + quarter (e.g., FY 2026-27 / Apr-Jun).
3. Click **GSTR-1 — Prepare Online** OR **Prepare Offline (use system JSON)**.
4. If offline route:
   - Open the GST offline utility (downloaded once).
   - Import the JSON exported by Synapsis Compliance OS.
   - Generate the upload file.
   - Back on portal → Upload → wait for validation green tick.
5. If online route (simpler for low invoice count):
   - Add each B2B invoice manually using data from `/admin/compliance/gst/gstr-1-prep`.
   - Add export invoices in 6A — Exports section.
   - Add B2C summary in section 7.
6. Click **Generate Summary** at bottom.
7. Compare with system's expected summary side-by-side.
8. **File** → OTP via registered mobile → ARN generated.

### After filing
- Save ARN PDF to Vault → `{fy}/gstr-1/gstr1_{quarter}_{ack}.pdf`.
- Update `tax_filings` row: kind=`gstr-1`, period, acknowledgement_ref, filed_at.

### Hidden gotcha
QRMP scheme requires you to **opt in** in the first month after GSTIN. If you don't opt in, default is monthly GSTR-1. System reminds to opt-in within first 30 days.

## 3. GSTR-3B Filing (monthly)

**When:** 20th of next month.
**Where:** Same portal → Returns Dashboard.
**Time needed:** 10 minutes.
**Why:** Summary return with self-assessed tax payment. Even under QRMP, GSTR-3B is monthly.

### Prerequisites — generated by the system
- Output tax (CGST/SGST/IGST) from all invoices in the month.
- ITC available (from `expenses` where `itc_eligible=true` + GSTR-2B reconciliation).
- Net tax payable.
- Challan amount.

### Steps
1. Login → **Returns Dashboard → GSTR-3B → Prepare Online**.
2. Section 3.1 — Outward supplies → fill from system:
   - 3.1(a) Taxable outward supplies → CGST + SGST + IGST values.
   - 3.1(b) Zero-rated exports → INR equivalent of USD export invoices.
3. Section 4 — Eligible ITC → fill from system (sum of `expenses.gst_amount_paise` where eligible).
4. Click **Save**.
5. Click **Compute Liability** → system shows tax to pay.
6. Pay via:
   - Use any **cash ledger balance** if available.
   - Else create challan: **Services → Payments → Create Challan** → CPIN generated → pay via net banking / UPI.
7. After challan paid, return to GSTR-3B → click **Make Payment / Post Credit to Ledger**.
8. **File GSTR-3B** → OTP → ARN generated.

### After filing
- Save ARN PDF to Vault → `{fy}/gstr-3b/3b_{yyyymm}_{ack}.pdf`.
- Update `tax_filings`.
- If challan paid: store challan PDF too.

### Late filing consequences
- ₹50/day late fee (₹25 CGST + ₹25 SGST).
- 18% p.a. interest on tax amount.
- System never lets this slip — T-7 / T-3 / T-1 reminders + dashboard red flag.

## 4. ITR-4 (Sugam) Filing (annual)

**When:** 31 July of assessment year (e.g., for FY 2026-27, deadline 31 Jul 2027). Usually extended to Oct.
**Where:** [Income Tax e-filing portal](https://www.incometax.gov.in).
**Time needed:** 30-60 minutes first time, 15 min after.
**Why:** Annual income tax return. ITR-4 is the simplified form for presumptive income (44ADA/44AD/44AE).

### Prerequisites — generated by the system
- Annual gross receipts INR-equivalent.
- Deemed income (50% of receipts).
- Deductions claimed (80C / 80CCD(1B) / 80D / 80TTA) with proof.
- TDS deducted by Indian clients (from Form 26AS — auto-fetched from portal).
- Bank account details (for refund, if any).
- Advance tax / self-assessment tax paid (with challan refs).

### Steps
1. Login → **e-File → Income Tax Returns → File Income Tax Return**.
2. Select Assessment Year, mode = Online, ITR-4 (Sugam).
3. Filing type: Section 139(1) original return.
4. **Personal Info** — pre-filled from PAN; verify address, contact, bank.
5. **Income Details**:
   - Income from business or profession → Section 44ADA → enter gross receipts and deemed income (50%).
   - Income from other sources → savings interest > ₹10k portion only.
6. **Deductions (Chapter VI-A)** — enter each section:
   - 80C — ₹X (specify PPF / ELSS / etc.)
   - 80CCD(1B) — ₹50,000 (NPS proof)
   - 80D — ₹25,000 (health insurance proof)
   - 80TTA — ₹10,000 (savings interest claim)
7. **Tax Paid** — confirm TDS (Form 26AS pre-populated) + advance tax challan.
8. **Verification** — compare system-computed tax vs portal-computed. Resolve any diff.
9. **Preview Return** → PDF.
10. **Submit** → e-verify via Aadhaar OTP / EVC / DSC.

### After filing
- Save acknowledgement (ITR-V) to Vault → `{fy}/itr/itr4_{fy}_ack.pdf`.
- Update `tax_filings` kind=`itr-4`.
- If refund expected → check status monthly on portal.

### Common pitfalls
- **Wrong section pick** — make sure it's 44ADA (specified profession), not 44AD (general business).
- **Receipts in foreign currency** — convert each invoice to INR using **SBI TT reference rate on date of receipt** (system pre-converts).
- **80CCD(2)** — employer NPS contribution; doesn't apply to proprietor. Leave blank.

## 5. Advance Tax Payment (annual under 44ADA)

**When:** 15 March of FY. Full 100% by this date.
**Where:** [NSDL TIN](https://onlineservices.tin.egov-nsdl.com) or income-tax portal.
**Why:** Required if total tax liability for FY exceeds ₹10,000.

### Steps
1. Open Challan 280 — [NSDL TIN site](https://onlineservices.tin.egov-nsdl.com/etaxnew/tdsnontds.jsp).
2. Select **(0021) Income Tax (Other than Companies)**.
3. Type of Payment → **(100) Advance Tax**.
4. Assessment Year → next FY.
5. Enter PAN, name, address.
6. Choose bank → pay via net banking → challan PDF.

### After payment
- Save challan PDF to Vault → `{fy}/advance-tax/challan_{date}.pdf`.
- Add row to `advance_tax_payments`.
- System reflects the credit in the running tax estimate.

## 6. NPS Tier 1 Account Opening (one-time)

**When:** Anytime; do it once you confirm 44ADA mode (so the ₹50k contribution actually saves tax).
**Where:** [eNPS by NSDL](https://enps.nsdl.com).
**Cost:** ₹0 to open. ₹200 minimum contribution per transaction.

### Steps
1. Open [enps.nsdl.com](https://enps.nsdl.com).
2. Click **National Pension System → Registration**.
3. Choose **Individual Subscriber**.
4. Aadhaar-based OR PAN+bank-based KYC. Aadhaar is faster (OTP route).
5. Fill personal details, bank, nominee.
6. Choose **Tier 1 only** (Tier 2 is optional liquid; we want only Tier 1 for the deduction).
7. Choose investment scheme:
   - **Auto Choice** → lifecycle fund (LC75 for aggressive at age 20).
   - **Active Choice** → set 75% E / 15% C / 10% G manually.
8. PRAN (Permanent Retirement Account Number) generated → save it.
9. Contribute initial amount (₹500 minimum). Subsequent contributions any time before 31 March.

### After opening
- Save PRAN PDF to Vault → `permanent/nps/pran.pdf`.
- Add to `deductions` upcoming for FY.
- Set yearly reminder: contribute ₹50k before 31 March.

## 7. Health Insurance Selection

**When:** Within 3-4 months (per user).
**Cost:** ₹8-15k/year for 20-yr-old comprehensive cover.

### Comparison (Indian individual policies, ₹5L cover)
| Insurer | Plan | Approx premium | Notes |
|---|---|---|---|
| HDFC ERGO | Optima Secure | ₹10-12k | Wide network, restoration benefit |
| Niva Bupa | ReAssure | ₹9-11k | Unlimited lock-in, claim history reset |
| Star Health | Comprehensive | ₹8-10k | Largest network in Maharashtra |
| Tata AIG | MediCare Plus | ₹11-13k | Strong claim ratio |

### What to look for
- **No room rent cap** (or very high cap).
- **No co-pay** at this age.
- **PED (pre-existing) waiting** — 2-3 years acceptable.
- **Restoration benefit** — auto-restore sum insured if exhausted.
- **Day-care procedures** covered (today's surgeries are mostly day-care).

### After purchase
- Save policy PDF to Vault → `permanent/insurance/health_{insurer}_{yyyy}.pdf`.
- Add to `deductions` kind=`80D` for current FY.
- Set yearly renewal reminder.

## 9. FIRC Retrieval (per international payment)

**When:** Within 30 days of each international payment.
**Where:** Your bank (whoever receives the Stripe payout) — usually the savings/current account bank.
**Why:** FIRC (Foreign Inward Remittance Certificate) is proof that the foreign exchange was received — required for GST export validity and FEMA compliance.

### Process
1. **Stripe payout lands** in your bank account.
2. Bank generates an inward remittance advice (FIRA) automatically — usually emailed.
3. For an official **FIRC** (stamped/signed), request via:
   - Bank's net banking → "Inward Remittance" / "Forex" section.
   - Or email RM with subject "FIRC request — remittance dated YYYY-MM-DD amount $X.XX".
   - Many banks issue FIRC only on request; some charge ₹100-300 per FIRC.
4. Bank emails/uploads PDF FIRC within 7 days.

### After receipt
- Save FIRC to Vault → `{fy}/firc/firc_{date}_${amount}.pdf`.
- Update `forex_inward` row with `firc_url` and `firc_reference`.
- Purpose code on FIRC should be **P0802** (computer software services / IT services). If wrong, write to bank to amend.

## 10. Current Account Opening

**Trigger conditions** (system alerts):
1. Monthly cashflow > ₹3L average for 3 months running.
2. Single client payment > ₹5L incoming.
3. Bank sends a query/notice about "high savings account turnover".
4. Need to issue cheques / bulk transfers under business name.

Until any of these fire → stay on savings. Per user decision 2026-05-13.

### When triggered — recommended banks
- **ICICI** — fastest onboarding for proprietorship, decent net banking.
- **HDFC** — premium NEFT/RTGS limits, best for clients who insist on private banks.
- **Kotak 811 Biz** — fully digital, no AMB for first year.
- **Axis Easy Business** — competitive.

### Documents (proprietorship)
- PAN (yours).
- Aadhaar (yours).
- Udyam Registration certificate.
- GST Registration certificate.
- Bank statement (savings, last 6 months).
- Address proof (electricity bill / rent agreement).
- Declaration: "I, Dhananjay Chitmilla, am the sole proprietor of Synapsis Industries."
- Trade name proof: Udyam + GST both mention "Synapsis Industries" → sufficient.

### After opening
- Update `business_tax_profile.bank_account_type = 'current'`, account details (server env vars).
- Update `synapsis-config.ts` to display current account.
- Migrate Razorpay settlement account + Stripe payout account to new current account.

## 11. GSTR-9 Annual Return

**When:** 31 December (for previous FY). **Only mandatory if turnover > ₹2 Cr.**

System alerts when turnover crosses ₹1.5Cr (T-50L buffer) so we have time to prep.

### Steps (high-level only — system will provide detailed flow when triggered)
1. Reconcile annual totals from monthly GSTR-3B and quarterly GSTR-1.
2. Reconcile ITC claimed vs GSTR-2B.
3. Reconcile any mismatch in HSN/SAC summaries.
4. File annual return + GSTR-9C reconciliation statement (if turnover > ₹5Cr, CA-certified).

For Synapsis at 20-50L: this is **not applicable**. Leave for the day it becomes relevant.
